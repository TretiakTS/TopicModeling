Methodology

My aim was to:

  • Retrieve all the articles from the given link
  • Clean the data
  • Build a corpus on it
  • Create the most efficient LDA model
  • Train it on the 80% of the corpus
  • Assign to each document a topic with the highest weight
  • Divide articles according to their dominant topics
  • Return the weights of words in each topic and the weight of topic in each document
  • Try the model in the unseen documents, assign created topics to them
  • Find the topic of the document
  • Return matrices of probability distributions, coherence and perplexity scores, number of documents with each topic .
  • Visualize the output for better understanding and analyzing
  • Analyze the results

As a result we can :

  • find hidden structures in documents, predominant words used in them and create topics
  • sort out a large amount of data according to its meaning, which helps in organizing and further use of documents
  • extract meaningful structures from texts, summarize them and predict what they can be about
  • evaluate our outputs.

In my final version of the program I used an NMF model

Non-Negative Matrix Factorization is a statistical method to reduce the dimension of the input corpora. It uses factor analysis method to provide comparatively less weightage to the words with less coherence.
